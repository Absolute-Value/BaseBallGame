## インストール | Installation

```console
pip install pygame
```

## 使用方法 | Usage

```console
python main.py
```

## 操作方法 | Controls

| キー | 説明 |
| --- | --- |
| w / up | 上へ移動 |
| s / down | 下へ移動 |
| a / left | 左へ移動 |
| d / right | 右へ移動 |
| N | 長押しでスイング、途中で離すことでバント |
